# Perfil: Freelancer

Vende tempo e talento pra clientes terceiros. O sistema gira em captar, entregar
e cobrar. Use este molde no `init` pra preencher `.businessos/memoria/`.

## Molde de `memoria/empresa.md`
```markdown
# Empresa
**Nome:** [nome]
**O que faz:** Freelancer de [design/dev/copy/marketing] — entrega [especialidade]
**Perfil:** Freelancer
**Atende clientes:** [tipo de cliente real]
**Serviços:** [principal 1, 2, 3]
**Capacidade:** [N projetos simultâneos no máx]   **Ticket médio:** [se quiser]
**Ferramentas:** [Notion, Gmail, Stripe... marcar conforme conectar MCP]
```

## Molde de `memoria/estrategia.md`
**Gargalo atual** · **Pra tirar das costas** · **Próximas prioridades**.

## Notas do perfil
- Organização por cliente é opcional e fica no projeto host (ex: pasta por cliente). O BusinessOS não impõe; instale-o na pasta de cada cliente se quiser memória separada, ou use uma só e registre os clientes na memória.
- `email`, `publicar-tema` (portfólio/LinkedIn), `analisar-dados` (relatórios de cliente) são os mais usados.
- Clientes ativos podem ser mantidos na memória; `atualizar` reconcilia.
