# Perfil: Solopreneur / criador solo

Uma pessoa, uma marca, audiência como ativo principal. O sistema gira em torno do
que você produz e publica. Use este molde no `init` pra preencher
`.businessos/memoria/`.

## Molde de `memoria/empresa.md`
```markdown
# Empresa
**Nome:** [nome / marca pessoal]
**O que faz:** [uma frase do jeito que falaria pro vizinho]
**Perfil:** Criador solo — marca pessoal + negócio
**Audiência:** [quem te acompanha hoje, perfil real, tamanho se relevante]
**O que produz:** [vídeos, carrosséis, curso/mentoria/ebook...]
**Posicionamento:** [sua tese; por que te escolhem vs. o próximo criador]
**Ferramentas:** [Notion, Canva, Meta Ads... marcar conforme conectar MCP]
```

## Molde de `memoria/estrategia.md`
**Gargalo atual** · **Pra tirar das costas** (candidata a `mapear-rotinas`) · **Próximas prioridades** (o que ataca o gargalo).

## Notas do perfil
- Conteúdo é o motor. `carrossel` e `publicar-tema` são os comandos mais usados.
- Lançamento de produto próprio é evento recorrente — registre na estratégia.
