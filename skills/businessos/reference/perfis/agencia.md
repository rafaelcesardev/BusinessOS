# Perfil: Agência / consultoria

Equipe pequena entregando pra múltiplos clientes em paralelo. Gira em torno de
proposta, atendimento e produção. Use este molde no `init` pra preencher
`.businessos/memoria/`.

## Molde de `memoria/empresa.md`
```markdown
# Empresa
**Nome:** [nome da agência]
**O que faz:** Agência de [marketing/design/conteúdo/consultoria de IA]
**Perfil:** Agência
**Atende clientes:** [perfil de cliente real]
**Serviços:** [serviço 1, 2, 3]
**Equipe:** [N pessoas, cada uma fazendo o quê]
**Capacidade:** [N clientes ativos simultâneos]
**Ferramentas:** [Notion, Gmail, Canva, Meta/Google Ads... conforme conectar MCP]
```

## Molde de `memoria/estrategia.md`
**Gargalo atual** · **Pra tirar das costas** · **Próximas prioridades**.

## Notas do perfil
- Multi-cliente: o BusinessOS instala **por pasta**. Pra memória separada por cliente, instale numa pasta por cliente (`cd cliente && businessos init`). Pra um workspace só, mantenha os clientes na memória e use `atualizar` pra reconciliar.
- Produção recorrente: `seo`, `anuncio-google`, `relatorio-ads`, `publicar-tema`, `carrossel`.
- Casos de sucesso e propostas: outputs ficam em `.businessos/saidas/` ou na pasta do cliente no host.
